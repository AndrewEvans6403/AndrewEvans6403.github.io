A 2d rogue like action and adventure platforming game. Created using the Zero Engine

This game was created my senior year of highschool while I was attending running start courses at digipen institute of technology. This was my final project for game 150 and still to this day is one of my favorite projects and I hold it close to my heart. This was created with 2 programmers, 2 artists, and 1 sound designer. We had about 4 and a half weeks to create this project.

After working on this project it really opened my eyes to how much I love game development, simulation, and programming in general. This is what sparked my passion for development and problem solving. Ever since I have been hooked on learning new technologies and working on new projects.
